# Baron's Bounties
A mod adding several vanilla+ weapons, a bounty board and 15 unique bounties all designed around the in-game region of Sentara. This mod was a submission for the Sentara comp.

This repository is public to serve as a reference for modders wanting to create level instances, add new bounties or create new bounty types.
